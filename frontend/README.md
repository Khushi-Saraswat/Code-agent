# AI Code Reviewer

AI Code Reviewer is a web application designed to help developers review and improve their code using AI. The application provides a user-friendly interface to input code and receive AI-generated reviews and suggestions.

## Features

- **Code Editor**: Write and edit your code in a built-in editor with syntax highlighting.
- **AI Code Review**: Get automated code reviews and suggestions powered by AI.
- **Copy Functionality**: Easily copy your code and review results to the clipboard.
- **Responsive Design**: Fully responsive layout that works on various screen sizes.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A modern frontend build tool that significantly improves the development experience.
- **Prism.js**: A lightweight, extensible syntax highlighter.
- **Axios**: A promise-based HTTP client for making API requests.
- **React Icons**: A collection of popular icons for React applications.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.
